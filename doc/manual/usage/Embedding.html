For embedding the viewer into other websites a single JavaScript file is available (ngl.embedded.min.js), fully containing the viewer sans the GUI code.

Example

The following code creates a viewer and loads the structure of PDB entry *1CRN* from RCSB. The result is seen on the right. For more information on how to control the viewer see the API reference, starting with the [page:Stage] class. The sources of the [page:Molecular-representations] and [page:Selection-language] documentation pages also serve as examples of embedding the viewer.
<!DOCTYPE html> <html lang="en"> <head> <meta charset="utf-8"> </head> <body> <script src="ngl.embedded.min.js"></script> <script> if ( ! Detector.webgl ) Detector.addGetWebGLMessage(); document.addEventListener("DOMContentLoaded", function() { NGL.mainScriptFilePath = "path/to/ngl.embedded.min.js"; NGL.init( function(){ var stage = new NGL.Stage( "viewport" ); stage.loadFile( "rcsb://1crn" ); } ); } ); </script> <div id="viewport" style="width:400px; height:300px;"></div> </body> </html>
