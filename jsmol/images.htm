This page demonstrates some relatively new capabilities in Jmol relating to incommensurately modulated structures (msCIF data). Some interesting features highlighted include easy manipulation of data arrays for tabulation, the animated GIFS created using the CAPTURE command, the dynamic measurements, and a few more goodies that you can find in plott.spt. Oh -- and the modulations being visualized as phased "vibrations." See Jmol.zip -- Bob Hanson Dec 12, 2013
